:::info 前置条件
- 本地已安装 Docker
- docker（版本>=20.0.0）以及docker-compose。
:::

## 快速部署

通过 Docker 快速部署 SagooIoT 服务端和前端。

通过官方提供的内部git仓库，获取SagooIoT的Docker部署文件。

拉取后，进入到docker-compose目录，执行`docker-compose up -d`命令，即可启动SagooIoT服务。

## 访问系统

访问SagooIoT管理界面

- 服务端：`http://localhost:8200`
