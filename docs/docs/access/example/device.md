# 添加设备

## 添加设备

1.登录SagooIOT物联网平台。
2.在左侧导航栏中，选择设备管理>设备，点击新增设备。
3.在新建弹框中，根据设备实际情况，按照页面提示填写信息，然后单击确定。
4，在所属产品中，找到刚刚创建的新产品。

## 启用设备

在设备列表中，找到刚刚创建的新设备，点击"启用"来启用这个设备。
