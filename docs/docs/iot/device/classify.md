# 产品分类

基于该功能实现对不同厂家、类型产品进行自定义分类管理，包含筛选、新增、列表等功能。创建自定义分类并设置名称、父级、排序等属性；展示所有分类信息，包括名称、父级、创建时间、更新时间等。

| 配置项  | 说明          |
|------|-------------|
| 上级分类 | 该分类的归属分类    |
| 分类标识 | 必填项，分类的标识信息 |
| 分类名称 | 必填项，分类的名称   |
| 排序   | 排序          |
| 描述   | 分类的说明信息     |
