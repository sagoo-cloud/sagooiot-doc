## 模拟设备

可以通过网络上下载的模拟设备进行测试。推荐使用[https://github.com/EasyDarwin/EasyGBD](https://github.com/EasyDarwin/EasyGBD)进行模拟设备测试。

运行后：

按下面图中标注的位置填写服务端信息。

![img.png](test01.png)


服务端信息填写完成后，点击“开始”按钮，即可开始模拟设备测试。
